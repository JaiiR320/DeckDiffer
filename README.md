# DeckDiff

Edit an MTG deck against an imported baseline and export the resulting list.

## Quick Start

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Features

- Import a baseline deck list
- Validate cards with Scryfall
- Add, remove, restore, and change quantities in one deck editor
- See diff-style row states for added, removed, and changed cards
- Export the resulting deck list from the export modal
